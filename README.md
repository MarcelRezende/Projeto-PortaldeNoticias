📰 Tech News

Uma landing page moderna de um portal de notícias sobre tecnologia, desenvolvida para praticar HTML, CSS, Grid Layout e Flexbox.

🚀 Tecnologias

- HTML5
- CSS3
- CSS Grid
- Flexbox

✨ Funcionalidades

- Destaque para notícia principal
- Categorias de tecnologia
- Barra de pesquisa
- Cards de notícias
- Layout responsivo
- Tema escuro
